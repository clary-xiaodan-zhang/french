G013 疑问句的倒装
简单倒装 重复主语倒装
规则：
  • 倒装的谓语和代词主语之间需加上连字符“-”，如果倒装的是名词主语，则不需要加
  • 如果有助动词，助动词与主语倒装
  • 前后单词如果以元音字母结尾和开头，则需加上发音辅助“-t-”
  • 有名词主语时，谓语动词和代词主语都需要和名词主语性数配合
一般疑问句 定义：没有疑问词
条件：
  • 一般疑问句，主语是代词

规则：
  • 谓语动词-代词主语 + 其他
  • 助动词-代词主语 + 谓语动词 + 其他
条件：
  • 一般疑问句，主语是名词
  • 一般疑问句，主语是代词，宾语前置

规则：
和简单倒装基本相同，但是要在前面加上名词主语：
  • 名词主语 + 谓语动词-代词主语 + 其他
  • 名词主语 + 助动词-代词主语 + 谓语动词 + 其他

特殊疑问句 定义：有疑问词
条件：
  • （必须）特殊疑问句，主语是代词
  • （可选）特殊疑问句，主语是名词，且谓语动词没有补语没有宾语

规则：
和一般疑问句基本相同，但介词+疑问词要放句首：
  • （介词 + ）疑问词 + 谓语动词-代词主语 + 其他
  • （介词 + ）疑问词 + 助动词-代词主语 + 谓语动词 + 其他
条件：
  • （必须）特殊疑问句，主语是名词，谓语动词有补语或有宾语
  • （可选）特殊疑问句，主语是名词，且谓语动词没有补语没有宾语

规则：
和一般疑问句基本相同，但介词+疑问词要放句首：
  • （介词 + ）疑问词 + 名词主语 + 谓语动词-代词主语 + 其他
  • （介词 + ）疑问词 + 名词主语 + 助动词-代词主语 + 谓语动词 + 其他
